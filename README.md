
# 班级作业查询系统

一个简单的班级作业查询系统，包含登录、作业查看、反馈、个人中心和管理员功能。

## 功能特点

- 登录系统：支持管理员和学生账号登录
- 今日作业：显示当天发布的作业
- 以往作业：以列表形式显示历史作业
- 管理员反馈：学生可以向管理员提交反馈
- 个人中心：修改昵称和密码
- 管理员功能：
  - 发布作业
  - 修改和删除作业
  - 查看学生反馈
  - 管理账号（添加、批量添加、删除）

## 默认账号

- 管理员账号：shareuser / user
- 学生账号：1-52，密码同账号

## 安装和运行

### 安装依赖

```bash
npm install
```

### 启动服务器

```bash
npm start
```

服务器将在 `http://localhost:3000` 启动，打开浏览器访问该地址即可使用。

## 数据存储

所有数据存储在以下 JSON 文件中：

- `users.json`：用户信息
- `homework.json`：作业信息
- `feedback.json`：反馈信息

## 注意事项

本系统用于班级内部使用，数据同步通过后端服务器实现，不使用浏览器本地存储。
